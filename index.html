MDShare

Editor Markdown colaborativo en tiempo real

Ver en GitHub Visitar web

Aplicación full-stack que permite a varios usuarios editar documentos Markdown simultáneamente con sincronización en vivo mediante WebSockets. Los documentos son privados: el propietario invita colaboradores por email y revoca el acceso cuando quiere. El backend cachea el estado en memoria y lo persiste cada 5 segundos.
